非標準の [`window.mozPaintCount`](https://developer.mozilla.org/docs/Web/API/Window/mozPaintCount) プロパティが Firefox 72 で廃止されました。この API は他のどのブラウザーにも実装されていないため、ブラウザー判別目的で使われていない限り互換性リスクは非常に低いはずです。
