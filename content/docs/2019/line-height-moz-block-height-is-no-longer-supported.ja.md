CSS `line-height` プロパティの非標準 `-moz-block-height` 値がウェブコンテンツから使用できなくなりました。これは行高を外側のブロックの高さに合わせることを可能にするものでしたが、本来 Firefox 内部での使用のみ意図していました。他のどのブラウザーにも実装されていないことから、互換性リスクは非常に低いはずです。
