現在では廃止予定となっている XML Binding Language (XBL) を要素と結び付けるため Firefox 内部で使用することを意図していた `-moz-binding` CSS プロパティがウェブコンテンツから使用できなくなりました。以前はウェブ上で Firefox を対象とした CSS ハックに使用可能でしたが、Firefox 67 以降では無視されます。
